# UI Mockups: Visual Query Builder

This document contains detailed ASCII/text mockups for the visual query builder interface.

---

## Overview

The query builder allows users to construct search vectors visually instead of requiring seed songs. The interface combines multiple interaction patterns for different user preferences.

## Accessibility Considerations

1. **Keyboard Navigation**: All controls accessible via Tab, adjustable via Arrow keys
2. **Screen Reader**: ARIA labels for all interactive elements
3. **Color Blind**: Use patterns/icons in addition to colors
4. **High Contrast Mode**: Ensure sufficient contrast ratios
5. **Touch Targets**: Minimum 44x44px for mobile touch targets
